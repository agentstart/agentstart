AgentStart 是一个 TypeScript AI Agent 框架。它开箱即用地提供了一整套工具和数据库适配器，使构建具有持久化内存、工具执行和类型安全客户端集成的生产级智能体变得容易。

## 为什么选择 AgentStart？

构建 AI 智能体通常需要拼凑多个库并为持久化、工具管理和客户端-服务器通信编写样板代码。AgentStart 通过提供以下内容解决了这个问题：

- **统一的智能体运行时**：基于 [AI SDK](https://ai-sdk.dev) 构建，使用 [oRPC](https://orpc.unnoq.com/) 实现类型安全通信
- **数据库适配器**：一流支持 [Drizzle](https://orm.drizzle.team/)、[Prisma](https://www.prisma.io/)、[Kysely](https://kysely.dev/) 和 [MongoDB](https://www.mongodb.com)
- **内置工具**：开箱即用的文件操作、Shell 命令和任务管理
- **React Hooks**：具有自动状态管理的类型安全客户端集成
- **框架集成**：为 Next.js、Node.js 等提供即用型适配器

## 快速示例

这是一个最小的 AgentStart 智能体示例：

```ts
// 服务端
import { Agent, innerTools, osTools } from "agentstart/agent";
import { agentStart } from "agentstart";
import { drizzleAdapter } from "agentstart/db";

const agent = new Agent({
  model: openrouter("x-ai/grok-4-fast"),
  instructions: "你是一个乐于助人的助手。",
  tools: { ...innerTools, ...osTools },
});

export const start = agentStart({
  agent,
  memory: drizzleAdapter(db, { provider: "pg", schema }),
  appName: "my-app",
});
```

```tsx
// 客户端
import { useThread } from "@/lib/agent-client";

export function Chat() {
  const { threadId } = useParams();
  useThread(threadId);
  
  return <ChatInterface />;
}
```
