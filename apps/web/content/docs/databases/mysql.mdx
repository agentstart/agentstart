# MySQL

MySQL is a popular open-source relational database management system (RDBMS) that is widely used for building web applications and other types of software. It provides a flexible and scalable database solution.

## Example Usage

Make sure you have MySQL installed and configured. Then, you can connect it directly to AgentStart.

```ts title="lib/agent.ts"
import { agentStart } from "agentstart";
import { Kysely, MysqlDialect } from "kysely";
import { createPool } from "mysql2/promise";

const db = new Kysely({
  dialect: new MysqlDialect({
    pool: createPool({
      host: "localhost",
      user: "root",
      password: "password",
      database: "agentstart",
    }),
  }),
});

export const start = agentStart({
  agent,
  memory: kyselyAdapter(db, {
    type: "mysql", // optional config
  }),
});
```

<Callout>
  For more information, read Kysely's documentation on the
  [MysqlDialect](https://kysely-org.github.io/kysely-apidoc/classes/MysqlDialect.html).
</Callout>

## Using with Drizzle

For a better development experience, we recommend using the [Drizzle adapter](/docs/databases/drizzle):

```bash
npm install drizzle-orm mysql2
```

```ts title="lib/agent.ts"
import { drizzle } from "drizzle-orm/mysql2";
import mysql from "mysql2/promise";
import { drizzleAdapter } from "agentstart/memory";

const pool = mysql.createPool({
  host: "localhost",
  user: "root",
  password: "password",
  database: "agentstart",
});

const db = drizzle(pool);

export const start = agentStart({
  agent,
  memory: drizzleAdapter(db, {
    provider: "mysql",
    schema: {
      thread: threadsTable,
      message: messagesTable,
    },
  }),
});
```

## Connection String Format

```bash
DATABASE_URL=mysql://user:password@localhost:3306/dbname
```

For cloud providers:

```bash
# PlanetScale
DATABASE_URL=mysql://user:password@aws.connect.psdb.cloud/database?ssl={"rejectUnauthorized":true}

# Railway
DATABASE_URL=mysql://root:password@containers-us-west.railway.app:3306/railway
```
