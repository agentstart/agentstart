AgentStart is a TypeScript AI agent framework. It ships with a full set of tools and database adapters out of the box, making it easy to build production-ready agents with persistent memory, tool execution, and type-safe client integration.

## Why AgentStart?

Building AI agents often requires piecing together multiple libraries and writing boilerplate code for persistence, tool management, and client-server communication. AgentStart solves this by providing:

- **Unified Agent Runtime**: Built on top of AI SDK with oRPC for type-safe communication
- **Database Adapters**: First-class support for Drizzle, Prisma, Kysely, and MongoDB
- **Built-in Tools**: File operations, shell commands, and task management out of the box
- **React Hooks**: Type-safe client integration with automatic state management
- **Framework Integration**: Ready-to-use adapters for Next.js, Node.js, and more

## Quick Example

Here's a minimal example of an AgentStart agent:

```ts
// Server
import { Agent, innerTools, osTools } from "agentstart/agent";
import { agentStart } from "agentstart";
import { drizzleAdapter } from "agentstart/db";

const agent = new Agent({
  model: openrouter("x-ai/grok-4-fast"),
  instructions: "You are a helpful assistant.",
  tools: { ...innerTools, ...osTools },
});

export const start = agentStart({
  agent,
  memory: drizzleAdapter(db, { provider: "pg", schema }),
  appName: "my-app",
});
```

```tsx
// Client
import { useThread } from "@/lib/agent-client";

export function Chat() {
  const { threadId } = useParams();
  useThread(threadId);
  
  return <ChatInterface />;
}
```
